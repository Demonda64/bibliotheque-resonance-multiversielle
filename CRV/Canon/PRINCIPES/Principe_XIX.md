# Principe XIX — Le Principe De Symetrie

Principe n°: XIX

Nom : Le Principe de Symétrie

Définition actuelle :
Toute connexion entre deux branches est potentiellement bidirectionnelle.

- Fondation IV : Gouvernance du Réseau
