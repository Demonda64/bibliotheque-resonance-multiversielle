# Principe VI — Tout existe quelque part

Principe n°: VI

Nom : Tout existe quelque part

Définition actuelle :
Tout ce qui est compatible avec les lois profondes du Champ Fondamental existe dans au moins une branche.

Fondation concernée :
- Fondation I : Le Multivers Infini

Questions :

1. Explique-t-il un phénomène fondamental ?
✓ Oui

2. Est-il irréductible ?
✓ Oui

3. Peut-il être déduit d'autres principes ?
□ Oui
✓ Non

4. Est-il une simple application ?
□ Oui
✓ Non

Décision :

✓ Principe fondamental
□ Corollaire
□ Théorème
□ Fusion

Justification :
Ce principe formalise l'ontologie du multivers : la pluralité des branches est le lieu d'existence des possibles compatibles. Il sert de socle métaphysique aux politiques de navigation et aux hypothèses de complétude du Canon.

Conséquences sur le Canon :
- Référencer ce principe dans les fiches sur création/validation d'artefacts et sur probabilités de branche.
- Vérifier cohérence avec `Principe X` (conservation de réalité) et `Principe II` (Noyau Identitaire).
