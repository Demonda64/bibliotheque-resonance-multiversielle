# Systeme Metrique de la Resonance

## Intention
Donner a la resonance multiversielle un langage quantifiable interne. Ces unites ne pretendent pas mesurer scientifiquement le monde physique; elles servent a rendre la cosmologie plus coherente, comparable et manipulable dans ses propres regles.

## Grandeurs Fondamentales

### Harmon (H)
Unite d'energie de convergence.

`1 H` represente la quantite minimale d'energie necessaire pour modifier durablement l'alignement d'une conscience vers une branche voisine.

Role: equivalent interne du Joule dans la cosmologie.

### Divergence (D)
Distance entre deux branches de realite.

`1 D` represente la difference minimale observable entre deux realites.

Exemples d'echelle:

| Divergence | Exemple |
| --- | --- |
| D = 1 | Prendre un cafe au lieu d'un the |
| D = 10 | Choisir une autre ecole |
| D = 100 | Exercer un autre metier |
| D = 1000 | Vivre dans un autre pays |
| D = 10000 | Devenir une personne radicalement differente |

### Reson (R)
Intensite instantanee de la connexion entre une conscience et une branche.

Echelle de travail:

| Reson | Etat |
| --- | --- |
| R = 0 | Aucune connexion |
| R = 10 | Pensee fugace |
| R = 100 | Visualisation intense |
| R = 1000 | Reve lucide |
| R = 10000 | Resonance exceptionnelle |

## Grandeurs Derivees

### Taux de Convergence (TC)
Quantite d'Harmons produits par unite de temps.

Mesure type: `H/jour`.

Exemples:

| Profil | TC indicatif |
| --- | --- |
| Individu moyen | 2 H/j |
| Meditant discipline | 10 H/j |
| Auteur obsessionnel | 25 H/j |
| Egregore collectif | 1000+ H/j |

### Coefficient de Coherence (C)
Valeur comprise entre `0` et `1`.

Mesure l'alignement entre:

- pensee;
- emotion;
- action.

Exemples:

| Situation | Coherence |
| --- | --- |
| Rever d'ecrire un livre sans jamais ecrire | C = 0,2 |
| Rever d'ecrire un livre et ecrire chaque jour | C = 0,9 |

Production reelle:

```text
H x C
```

### Masse de Realite (MR)
Resistance naturelle d'une branche au changement ou a la convergence.

Exemples:

| Branche cible | MR indicative |
| --- | --- |
| Finir un chapitre | MR = 10 |
| Changer de metier | MR = 100 |
| Devenir auteur reconnu | MR = 1000 |
| Devenir president d'un pays | MR = 10000 |

## Equation Fondamentale

```text
Convergence = (H x C x R) / D
```

Ou:

- `H` = Harmons produits;
- `C` = coherence pensee-emotion-action;
- `R` = intensite de resonance;
- `D` = divergence entre la branche actuelle et la branche cible.

## Exemple
Oscar veut terminer le chapitre 1.

```text
D = 50
H = 20
C = 0,9
R = 30

Convergence = (20 x 0,9 x 30) / 50
Convergence = 10,8
```

La convergence est forte, car la branche cible reste proche et coherente avec les actions d'Oscar.

Oscar veut devenir le penseur le plus influent du siecle avec la meme energie, la meme coherence et la meme resonance.

```text
D = 50000
H = 20
C = 0,9
R = 30
```

La convergence devient tres faible. La branche cible est trop lointaine pour un saut direct.

## Branches-Ponts
Lorsqu'une branche cible est trop divergente, la conscience doit passer par des branches intermediaires.

Exemple:

```text
Terminer un chapitre
-> terminer un livre court
-> publier un premier essai
-> creer une communaute de lecteurs
-> devenir une reference dans un domaine
```

Les branches-ponts reduisent la divergence effective en remplacant un saut impossible par une sequence de convergences possibles.

## Methode de Convergence Progressive
Voir aussi: `METHODE_CONVERGENCE_PROGRESSIVE.md`.

La methode de convergence progressive applique les metriques a une pratique:

- choisir une branche proche pour reduire `D`;
- visualiser le quotidien de cette branche pour stabiliser `R`;
- produire une action immediate pour generer des `H`;
- aligner pensee, emotion et action pour augmenter `C`;
- decomposer une cible a forte `MR` en branches-ponts.

Principe directeur:

```text
Une branche est atteinte non lorsqu'elle est desiree, mais lorsqu'elle devient familiere.
```

## Regle Editoriale
Les metriques doivent etre utilisees comme langage interne de la theorie, pas comme promesse de controle absolu. Elles doivent clarifier les couts, distances et dynamiques, tout en gardant la prudence speculative du projet.
